Установка:
Требуется Lua54, LuaRT для создания и компиляции игры

Создание:
Все скрипты используют формат .wlua а не .lua
Файлы игры находятся в папке game
Файл start.wlua это первый запускаемый файл при запуске приложения
Файл gameinfo.json это настройка приложения, название, иконка и т.д
Файл settings.json это настройки woowzengine, там можно включить консоль и т.д

Компиляция:
Запустите файл woowzengine_compile.bat, появиться терминал где будет расписана компиляция вашей игры, если произойдёт ошибка то .exe файл вашей игры не будет создан.
